## How to Use This Document

This document contains the full approved website copy for exl.agency. It is organized as the pages will be built and shipped. Each page includes every section, heading, paragraph, CTA, and micro-copy element needed for handoff to a designer or developer.

The document does not include design direction. Visual treatment, typography, color application, motion, and layout decisions are left to the designer. The only constraints are the EXL brand guidelines and the content below.

Placeholder CTAs appear in square brackets, like `[ Book a call ]`. These are button labels, not copy to be rendered as text on the page.

Capitalized words in small type before a heading, like `FEATURED SERVICE`, are section labels. They are display elements, not page titles.
